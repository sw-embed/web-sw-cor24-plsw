Added collapsible macro expansion view: preprocessor.rs parses MACRODEF blocks from .msw files and expands ?MACRO(...) invocations with parameter substitution into GEN block assembly lines. MacroExpansionView component renders preprocessed source with clickable toggle arrows on each invocation, showing/hiding expanded assembly with syntax coloring. Collapsed by default with line count badges. Wired into Preprocess wizard step with separate boot output cell. 3 unit tests for parser, expander, and full preprocessor.
